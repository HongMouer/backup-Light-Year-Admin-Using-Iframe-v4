#### 介绍
该项目是光年后台管理模板(Light Year Admin Iframe)基于bootstrap 4.4.1版本。

这里就不弄非iframe版本的了，该项目会进行一段时间，基于bootstrap 4.4.1版本。
因4.*的版本，一些class的命名已经改变，并且布局由float改成了flex，所以项目不支持从3.7版本升级到4.4.1版本。
